Criar um container com as configurações do docker-compose.yml
docker-compose up -d

Deletar o container criado anteriormente
docker-compose down -v


# O '--build' força o Docker a ler o Dockerfile de novo e instalar o psycopg2
docker-compose up -d --build

Entre no container e rode o app:
docker exec -it app-ia streamlit run app.py

navegar "dentro" do container pelo "Shell"
docker exec -it container-2 bash

-it: É o que permite a interatividade (seu teclado conectado ao Linux).

bash: É o nome do "terminal" Linux.

Uma vez lá dentro, o seu prompt vai mudar (ex: root@id-do-container:/app#). Agora você pode usar os comandos de navegação:

ls: Lista os arquivos da pasta atual.

cd /: Vai para a raiz do sistema Linux.

cd /app: Volta para a pasta onde está seu projeto.

pwd: Mostra exatamente em qual pasta você está.

exit: Sai do container e volta para o Windows.


Crie um repositorio no git primeiro

# 1. Inicializa o Git
git init

# 2. Adiciona todos os arquivos (O .gitignore vai barrar o .env automaticamente)
git add .

# 3. Cria o primeiro commit
git commit -m "Projeto 3: App IA com Postgres e Docker Compose"

# 4. Renomeia a branch para main (padrão atual)
git branch -M main

# 5. Adiciona o endereço remoto (Ajuste para o seu link SSH)
git remote add origin git@github.com:BrunoKaram/ai-chat-docker-orchestration.git

# 6. Envia para o GitHub
git push -u origin main
